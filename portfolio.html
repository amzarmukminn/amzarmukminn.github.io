Project 4 — Stationary Payment System

Personal Project — Stationary Payment System (Python Terminal & Single HTML)

Description:
A terminal-based with GUI point-of-sale system for a stationary shop, built entirely in Python & integrated to HTML. The system handles product determination, customer purchase input, total calculation with discount, and receipt generation through a clean command-line interface at first. Demonstrates strong understanding of Python logic, loops, functions, and data handling.

Key features:

  • Product item display with prices
  • Customer purchase selection and quantity input
  • Automatic total & change calculation
  • Terminal receipt generation on purchase completion
  • Refined the system to HTML with CSS and Javascript

Tools & Language: Python 3, CLI, GUI (CSS w/ Tailwind Framework & Lucide Icons)

Year: 2026 • Type: Personal Project
